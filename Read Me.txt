This project develops an image classification model using a Convolutional Neural Network (CNN) to classify natural scene images into six categories: Buildings, Forest, Glacier, Mountain, Sea, and Street. The Intel Image Classification dataset is used, with all images resized to 128 × 128 pixels and normalized to improve training. The dataset is then divided into training and validation sets, and TensorFlow's data pipeline is used for efficient loading and processing.

The model consists of multiple convolutional layers that automatically learn important features from the images. Techniques such as Batch Normalization, Dropout, and Data Augmentation (flipping, rotation, zooming, and contrast adjustment) are used to improve accuracy and reduce overfitting. The model is trained using the Adam optimizer and evaluated using accuracy, loss, a confusion matrix, and a classification report. The same workflow can also be extended to Transfer Learning with pretrained models like MobileNetV2 or EfficientNet for improved performance.

To run the project, install the required Python libraries, download and extract the Intel Image Classification dataset, and update the dataset path if needed. Run the notebook sequentially to preprocess the data, build and train the CNN model, evaluate its performance, and save the trained model for future use.
